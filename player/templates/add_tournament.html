There are two different options to add tournaments data. One automatically via a csv file and another one manually via a web interface.

Automatic

The idea is to create csv file, send it to us and we will just run our app with your csv file as input and the tournament data will automatically online at the website.

A file can be created with your favorite text editor but also with a more comfortable spreadsheet software like Excel, LibreOffice or OpenOffice.

The csv file must have an specific format otherwise the software will not allow to upload your tournament data to the web.

The most practical approach to create your own csv file is to download this file which has been already used by us to add tournaments data. After download it, open and edit it adapting the column values to your own tournament data.

A line contains all the information of a tournament game and it must look like the following examples (for more examples download this file):

World Cup 2015;MO;04/29/15;11:30;Field 2;Pool A;Gold;4;xx;France;8;5;Malaysia
World Cup 2015;MO;04/30/15;12:20;Field 15;Division;Silver;8;xx;England;8;3;Wales
World Cup 2015;MO;05/03/15;12:30;Field 3;Fifth position;Gold;2;xx;Japan;5;3;Fiji
World Cup 2015;M30;05/02/15;20:00;Field 5;1/4;Gold;2;xx;Fiji;3;4;Papua New Guinea
World Cup 2015;M30;05/03/15;08:00;Field 1;Semifinal;Gold;2;xx;Australia;16;0;Papua New Guinea
World Cup 2015;MO;05/03/15;18:40;Field 1;Final;Gold;2;xx;Australia;11;2;New Zealand


This is the meaning of each column or comma separated value:

  1. Name of the tournament
  2. Division, e.g: MO, WO, MXO, M30, W27, SMX, etc...
  3. Date of the game
  4. Field name
  5. Phase of the tournament: Pool A, Division, Final, Ninth position, 1/4, Semifinal, etc...
  6. Category: which kind of cup is the team playing for Gold, Silver, Bronze or Wood
  7. Numbers of teams playing for the given phase and category.
  8. xx: this value is constant
  9. Local team name
  10. Local team score
  11. Visitor team score
  12. Visitor team name

If you want to add the statistics of the tournament you need to create a separated csv file with a different format. You can follow the same procedure, download this file and edit it. The format of the statistic file look like (for more examples download this file):

World Cup 2015;MO;Fiji;5;Taito;Nawaituruturu;M;0;South Africa;7;3;Fiji;Gold;Division;8 World Cup 2015;MO;Australia;4;Ben;Moylan;M;4;Australia;11;2;New Zealand;Gold;Final;2 World Cup 2015;MO;England;12;Benjamin;Meakin;M;0;New Zealand;11;2;England;Gold;Pool B;4 World Cup 2015;MO;United States of America;15;Malik;Vano;M;1;United States of America;5;2;Singapore;Silver;Division;8


This is the meaning of each column or comma separated value:

  1. Name of the tournament
  2. Division, e.g: MO, WO, MXO, M30, W27, SMX, etc...
  3. Scorer team
  4. Number of the scorer player
  5. First name of the scorer player
  6. Last name of the scorer player
  7. Gender of the scorer player: M, F, U (Unknown)
  8. Total touch downs of the player, it could be 0
  9. Local team name
  10. Local team score
  11. Visitor team score
  12. Visitor team name
  13. Category: which kind of cup is the team playing for Gold, Silver, Bronze or Wood
  14. Phase of the tournament: Pool A, Division, Final, Ninth position, 1/4, Semifinal, etc...
  15. Numbers of teams playing for the given phase and category.


Once you have your csv file ready please contact us and we will upload your tournament data. We are also please to help you to create your csv file or answer your questions.

Manually

This app is developed with django which offer an administration interface where you can add via web forms tournaments, games, players, persons, etc...

You should request a user name and password to login in into the administration interface, if you want to use this process to put your tournament data online please us.

If you are interested in how does the administration interface looks and works you can watch this youtube video.
